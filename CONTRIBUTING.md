# Contributing to News Magazine

First off, thank you for considering contributing to this project!

## How to Contribute

### Reporting Issues
If you find a bug or have a suggestion for a new feature, please open an issue in the **Issues** tab.

### Pull Requests
If you want to help **Mahoor** improve the code or the news magazine's layout:
1. Fork the repository.
2. Create a new branch for your changes.
3. Submit a **Pull Request** explaining what you have changed.

Your help is greatly appreciated!
