# Security Policy

## Supported Versions

Currently, only the latest version of this project is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest| :x:                |

## Reporting a Vulnerability

I, **Mahoor**, take the security of this project and the users who run this script seriously.

If you discover a potential security vulnerability (such as a bug that could expose GitHub Actions secrets or compromise the environment), please **do not open a public Issue**.

Instead, please report it through one of the following ways:

*   **Email:** Send an email to [Mojtabatavousi136@gmail.com](mailto:Mojtabatavousi136@gmail.com)
*   **Private Vulnerability Reporting:** You can use GitHub's private reporting feature if enabled for this repository.

### What to expect:
*   I will acknowledge your report within 48-72 hours.
*   I will provide a fix as soon as possible for any confirmed vulnerabilities.
*   You will be credited for the discovery in the release notes unless you prefer to remain anonymous.

Thank you for helping keep this open-source project safe!
